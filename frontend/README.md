# Frontend - Task Manager

Interface web desenvolvida com React, TypeScript e Vite.

## Instalação
```bash
npm install
```

## Executar
```bash
# Desenvolvimento
npm run dev

# Build
npm run build

# Preview do build
npm run preview
```

## Acessar

http://localhost:5173

## Funcionalidades

- Criar tasks com validação
- Listar tasks em grid responsivo
- Feedback visual de loading
- Tratamento de erros
